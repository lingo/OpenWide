About

OpenWide was written to avoid a few [more] minor annoyances with Windows.
This program allows you to specify the position and size of Windows' Open and
Save dialog boxes, and also to specify where the initial focus should be and
which view will be the default.

Good luck with it and enjoy.

This is a test release. Limited testing has shown it to work on a release of
Win2000 Professional and a release of WinXP Home.

Features

  • Set position and size of Open & Save dialogs
  • Set initial keyboard focus
  • Set initial view style
  • Drag & drop folders onto the title bar of an Open/Save dialog box,
    and the view will change to that folder!

Help

OpenWide Main Window

Here you see the main window for OpenWide. If this window doesn't appear when you first load OpenWide, look for the icon in the windows taskbar and click on it with the left mouse button.

Settings

After changing any setting, please click the Apply button before clicking Test or minizing the window.

Open & Save Dialogs
Test Current Settings
Click here to test the current settings. Make sure to click Apply first.
Left, Top, Width, Height
You can set the position and size of the Open/Save dialogs here, or click the Set button to do so interactively.
Set
This will show you a window. Move and size this window to set the position and size of the Open/Save dialog boxes.
Focus on:
Here you can choose which area or control will initially have the input focus when an Open or Save dialog box is shown. If a control has the input focus, this means that keyboard input will be sent to it.
View:
Here you can choose how you will view the list of files & folders in the Open/Save dialog boxes. You may choose one of the Windows standard view settings.
Exclusions

The programs which are listed in this section will not have their Open & Save dialogs modified by OpenWide.

Add program
Click this button, then find the application file you wish to exclude.
Remove program
Just as it says, will remove a program from the list of excluded.
Preferences

Note that all settings for OpenWide are per-user settings. This means that they will only apply to the user who set them up.

Start minimized
If this is checked, then the main window will not be shown when OpenWide starts up. Only the icon will be shown in the system tray on the taskbar.
Start with Windows
If this is checked, a shortcut to OpenWide will be placed into the Startup folder on the Start Menu. This will make OpenWide start automatically with Windows.
Show icon in system tray.
If this is not checked, no icon will show in the Windows taskbar. To configure OpenWide when no icon is showing, simply run the program again. This will cause the main window to be displayed.

Open/Save Dialog features

Locate folder
Right click on the window title-bar to get the menu displayed above. Selecting "Locate current folder...." will open an Explorer window with the current folder selected.
Drag and Drop
Drag and drop a folder onto the Open or Save dialog box title-bar to change to that directory.
